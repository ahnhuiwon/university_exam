# 5강 - 보조기억장치와 데이터 저장

## 1. 보조기억장치 ⭐⭐⭐ 시험 핵심

### 의미
> 주기억장치를 보조하는 2차 기억장치

### 역할
- 대용량 데이터 저장
- 비휘발성 데이터 저장

---

# 2. 주기억장치와 보조기억장치 차이 ⭐⭐⭐ 시험 핵심

| 구분 | 주기억장치 | 보조기억장치 |
|---|---|---|
| 속도 | 빠름 | 느림 |
| 용량 | 작음 | 큼 |
| 가격 | 비쌈 | 저렴 |
| 휘발성 | 휘발성 | 비휘발성 |

---

# 3. 휘발성 기억장치 ⭐⭐⭐ 중요

### 특징
> 전원이 꺼지면 데이터 소멸

### 예시
- RAM

---

# 4. 비휘발성 기억장치 ⭐⭐⭐ 시험 핵심

### 특징
> 전원이 꺼져도 데이터 유지

### 예시
- HDD
- SSD
- USB

---

# 5. 보조기억장치 필요성 ⭐⭐⭐ 시험 핵심

### 이유
- 대용량 데이터 저장
- 데이터 영구 보관
- 주기억장치 보조

---

# 6. 지역성(Locality) 원리 ⭐⭐⭐ 중요

### 의미
> 전체 데이터가 아닌 일부 데이터만 반복 사용

---

# 7. 기억장치 계층구조 ⭐⭐⭐ 시험 핵심

### 구성 순서
```text
레지스터 → 캐시 → 주기억장치 → 보조기억장치
```

### 특징
- 위로 갈수록 빠름
- 아래로 갈수록 용량 큼

---

# 8. 캐시 메모리 ⭐⭐⭐ 시험 핵심

### 역할
> CPU와 주기억장치 속도 차이 완화

---

# 9. 접근(Access) ⭐⭐⭐ 중요

### 의미
> 데이터를 읽거나 쓰는 행위

---

# 10. 순차접근(Sequential Access) ⭐⭐⭐ 시험 핵심

### 특징
- 앞에서부터 차례대로 접근
- 접근 속도 느림

### 예시
- 자기테이프

---

# 11. 직접접근(Direct Access) ⭐⭐⭐ 시험 핵심

### 특징
- 원하는 위치 바로 접근 가능

### 예시
- HDD
- SSD
- 광디스크

---

# 12. 자기테이프 ⭐⭐⭐ 시험 핵심

### 특징
- 순차접근 저장장치
- 대용량 백업용

---

# 13. 자기테이프 용도 ⭐⭐⭐ 중요

### 활용
- 데이터 백업
- 장기 보관

---

# 14. 릴 테이프 ⭐⭐⭐ 중요

### 특징
- 초기 자기테이프 방식

---

# 15. 카트리지 테이프 ⭐⭐⭐ 시험 핵심

### 특징
- 보호와 관리 편리

---

# 16. DAT ⭐⭐⭐ 중요

### 의미
```text
Digital Audio Tape
```

---

# 17. LTO ⭐⭐⭐ 시험 핵심

### 의미
```text
Linear Tape Open
```

### 특징
- 대용량 저장 가능

---

# 18. 테이프 라이브러리 ⭐⭐⭐ 중요

### 특징
- 로봇이 자동 장착
- 초대용량 저장

---

# 19. 자기디스크 ⭐⭐⭐ 시험 핵심

### 특징
- 직접접근 저장장치
- 원판 회전 방식

---

# 20. 자기디스크 구성 ⭐⭐⭐ 시험 핵심

### 구성요소
- 디스크
- 헤드
- 액추에이터 암

---

# 21. 플로피디스크 ⭐⭐⭐ 중요

### 특징
- 휴대형 자기디스크
- 현재는 거의 사용 안함

---

# 22. 저장 아이콘 ⭐⭐⭐ 중요

### 특징
> 플로피디스크 모양 사용

---

# 23. 하드디스크(HDD) ⭐⭐⭐ 시험 핵심

### 특징
- 금속 디스크 사용
- 고속 회전 방식

---

# 24. 액추에이터 암 ⭐⭐⭐ 중요

### 역할
> 헤드를 이동시키는 장치

---

# 25. 트랙(Track) ⭐⭐⭐ 시험 핵심

### 의미
> 원형 데이터 저장 경로

---

# 26. 섹터(Sector) ⭐⭐⭐ 시험 핵심

### 의미
> 데이터 저장 기본 단위

---

# 27. 실린더(Cylinder) ⭐⭐⭐ 중요

### 의미
> 동일 반경 트랙 집합

---

# 28. 탐색시간(Seek Time) ⭐⭐⭐ 시험 핵심

### 의미
> 헤드가 원하는 트랙으로 이동하는 시간

---

# 29. 회전지연시간 ⭐⭐⭐ 시험 핵심

### 의미
> 디스크 회전으로 데이터가 헤드 위치까지 오는 시간

---

# 30. 전송시간 ⭐⭐⭐ 중요

### 의미
> 데이터를 메모리로 전송하는 시간

---

# 31. RPM ⭐⭐⭐ 중요

### 의미
```text
분당 회전 수
```

---

# 32. 디스크 어레이 ⭐⭐⭐ 시험 핵심

### 특징
- 여러 디스크를 배열
- 성능/신뢰성 향상

---

# 33. RAID ⭐⭐⭐ 시험 핵심

### 의미
```text
Redundant Array of Independent Disk
```

### 목적
- 데이터 안정성
- 속도 향상

---

# 34. 광디스크 ⭐⭐⭐ 시험 핵심

### 특징
- 레이저 사용
- 직접 접근 가능

---

# 35. ROM 방식 광디스크 ⭐⭐⭐ 시험 핵심

### 특징
- 읽기 전용

### 예시
- CD-ROM

---

# 36. WORM 방식 ⭐⭐⭐ 중요

### 의미
```text
Write Once Read Many
```

### 특징
- 한 번만 기록 가능

---

# 37. RW 방식 ⭐⭐⭐ 시험 핵심

### 특징
- 반복 기록 가능

---

# 38. CD ⭐⭐⭐ 시험 핵심

### 의미
```text
Compact Disc
```

### 용량
- 약 650~700MB

---

# 39. DVD ⭐⭐⭐ 시험 핵심

### 의미
```text
Digital Versatile Disc
```

### 용량
- 4.7GB
- 듀얼레이어 8.5GB

---

# 40. 블루레이 ⭐⭐⭐ 시험 핵심

### 특징
- 청색 레이저 사용
- 고용량 저장

### 용량
- 25GB ~ 50GB

---

# 41. CLV ⭐⭐⭐ 중요

### 의미
```text
Constant Linear Velocity
```

### 특징
- 선속도 일정
- 나선형 트랙 구조

---

# 42. CAV ⭐⭐⭐ 중요

### 의미
```text
Constant Angular Velocity
```

### 특징
- 회전속도 일정
- 직접접근 유리

---

# 43. USB Flash Drive ⭐⭐⭐ 시험 핵심

### 특징
- 플래시 메모리 기반

---

# 44. Flash Memory ⭐⭐⭐ 시험 핵심

### 특징
- 비휘발성 반도체 메모리

---

# 45. USB ⭐⭐⭐ 중요

### 의미
```text
Universal Serial Bus
```

---

# 46. SLC ⭐⭐⭐ 시험 핵심

### 의미
```text
Single Level Cell
```

### 특징
- 1셀 1비트 저장
- 빠르고 안정적
- 비쌈

---

# 47. MLC ⭐⭐⭐ 시험 핵심

### 의미
```text
Multi Level Cell
```

### 특징
- 여러 비트 저장
- 저렴함

---

# 48. TLC ⭐⭐⭐ 중요

### 특징
- 1셀 3비트 저장

---

# 49. QLC ⭐⭐⭐ 중요

### 특징
- 1셀 4비트 저장

---

# 50. SSD ⭐⭐⭐ 시험 핵심

### 의미
```text
Solid State Drive
```

### 특징
- 플래시 메모리 기반
- 기계적 움직임 없음

---

# 51. SSD 장점 ⭐⭐⭐ 시험 핵심

### 특징
- 속도 빠름
- 소음 적음
- 전력 소모 적음
- 충격 강함

---

# 52. SSD 단점 ⭐⭐⭐ 중요

### 특징
- 가격 비쌈
- 수명 제한

---

# 53. Wear Leveling ⭐⭐⭐ 중요

### 의미

---

# 54. SSD 특징 ⭐⭐⭐ 중요

### 특징

- 쓰기 속도는 상대적으로 느림

# 56. 클라우드 저장장치 ⭐⭐⭐ 시험 핵심
