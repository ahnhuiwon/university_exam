# 6강 - 소프트웨어와 정보 시스템

## 1. 소프트웨어 ⭐⭐⭐ 시험 핵심

### 의미
> 컴퓨터가 동작하도록 하는 프로그램과 관련 데이터

### 특징
- 하드웨어 제어
- 명령 수행
- 사용자 작업 지원

---

# 2. 소프트웨어 분류 ⭐⭐⭐ 시험 핵심

| 구분 | 설명 |
|---|---|
| 시스템 소프트웨어 | 컴퓨터 운영 지원 |
| 응용 소프트웨어 | 사용자의 작업 수행 |

---

# 3. 시스템 소프트웨어 ⭐⭐⭐ 시험 핵심

### 역할
- 하드웨어 관리
- 응용 소프트웨어 지원
- 컴퓨터 운영

### 예시
- 운영체제
- 언어 번역기
- 유틸리티

---

# 4. 응용 소프트웨어 ⭐⭐⭐ 시험 핵심
