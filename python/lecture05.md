# 5강 - 선택 구조

## 1. 선택 구조(Selection Structure) ⭐⭐⭐ 매우 중요

### 중요도
⭐⭐⭐⭐⭐

### 왜 중요한가?
- 구조적 프로그래밍의 핵심 구조
- 프로그램의 "지능의 시작"이라고 강조
- 시험 핵심 개념 가능성 높음

### 핵심 정리
- 조건에 따라 실행 여부를 결정하는 구조
- 프로그램 흐름을 분기시키는 역할

### 정의
> 조건의 참/거짓에 따라 특정 명령문의 실행 여부를 결정하는 구조

### 시험 포인트
- 구조적 프로그래밍의 핵심 제어 구조
- 분기(Branch)를 만드는 역할

---

### 중요도
⭐⭐⭐⭐

### 핵심 정리

## 4. if문 ⭐⭐⭐ 시험 핵심

### 중요도
⭐⭐⭐⭐⭐

### 기본 형식

```python

    실행문
````

### 핵심 정리

* 조건이 True일 때만 실행
* 조건식 뒤에는 반드시 `:` 사용

* if + 콜론(:)
* 들여쓰기 필수

## 18. 단락 평가(Short Circuit Evaluation) ⭐⭐⭐ 시험 가능성 높음

### 중요도

⭐⭐⭐⭐⭐

### 핵심 정리

일부 결과만으로 전체 결과가 결정되면
뒤 연산 생략

---

### and 단락 평가

```python
False and ??
```

→ 뒤 계산 안 함

---

### or 단락 평가

```python
True or ??
```

→ 뒤 계산 안 함

### 시험 포인트

* 실행 속도 최적화 기법
