# 7강 - 함수(Function)

## 1. 함수(Function) ⭐⭐⭐ 매우 중요

### 중요도
⭐⭐⭐⭐⭐

### 왜 중요한가?
- 객체지향(OOP)의 시작점
- 프로그램 재사용성과 유지보수 핵심

### 핵심 정리
> 특정 작업을 수행하는 명령문의 집합

### 특징
- 기능을 이름으로 호출 가능
- 내부 복잡성을 숨김
- 필요한 값만 전달받아 처리

### 시험 포인트
- 함수 = 추상화의 시작
- 내부 동작 숨김 → 인캡슐레이션 개념 연결

---
