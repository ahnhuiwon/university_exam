# 6강 - 반복 구조

## 1. 반복 구조(Loop / Iterate / Repeat) ⭐⭐⭐ 매우 중요

### 중요도
⭐⭐⭐⭐⭐

### 정의
> 특정 명령문을 여러 번 반복 실행하는 구조

### 핵심 정리
- 반복 작업 자동화
- 구조적 프로그래밍의 핵심
- 선택 구조 + 반복 구조 = 프로그램 핵심

### 시험 포인트
- 반복 관련 용어
  - Loop
  - Iterate
  - Repeat

---

## 2. 유클리드 호제법 ⭐⭐⭐ 중요

### 중요도
⭐⭐⭐⭐⭐

### 핵심 정리
- 최대공약수(GCD)를 구하는 알고리즘
- 반복 구조 대표 사례

### 원리
1. 큰 수를 작은 수로 나눔
2. 나머지를 구함
3. 작은 수와 나머지로 반복
4. 나머지가 0이 되면 종료
5. 마지막 제수가 최대공약수

### 중요도
⭐⭐⭐⭐

### 핵심 정리
- 같은 작업 반복

- 조건에 따라 반복 횟수 결정

### 시험 포인트
- 반복 횟수는 조건식(Boolean)으로 결정

## 4. 반복 구조 종류 ⭐⭐⭐ 시험 핵심

## 종류

| 종류 | 설명 |

# 5. while문 ⭐⭐⭐ 매우 중요

### 중요도
⭐⭐⭐⭐⭐

### 형식

```python
while 조건식:
    실행문
```

### 핵심 정리
- 조건이 True인 동안 반복
- False가 되면 종료

### 시험 포인트
- while 뒤에는 Boolean 식
- 콜론(:) 필수
- 들여쓰기 필수

---

## 6. while 반복 흐름 ⭐ 중요

### 중요도
⭐⭐⭐⭐⭐

### 실행 순서
1. 조건 검사
2. True → 실행
3. 다시 조건 검사
4. False → 종료

## 7. 반복 출력 프로그램 ⭐ 중요

⭐⭐⭐⭐

# 14. n까지 합 구하기 ⭐⭐⭐ 시험 핵심

### 중요도
⭐⭐⭐⭐⭐

### 코드

```python
n = int(input("숫자 입력: "))

sum = 0
i = 1

while i <= n:
    sum += i
    i += 1

print(sum)
```

### 핵심 정리
- 누적 변수 필요
- 반복 변수 증가 필요

# 29. 최대공약수 프로그램 ⭐⭐⭐ 시험 가능성 높음

### 중요도
⭐⭐⭐⭐⭐

### 코드

```python
a = int(input())
b = int(input())

while b != 0:
    r = a % b
    a = b
    b = r

print(a)
```

### 시험 포인트
- 종료 조건
- 나머지 계산
- 마지막 a 출력
