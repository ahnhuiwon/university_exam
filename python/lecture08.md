# 2. 컬렉션(Collection) ⭐⭐⭐ 시험 핵심

### 중요도
⭐⭐⭐⭐⭐

### 정의
> 여러 데이터를 묶어 저장하는 자료구조

### 종류

| 종류 | 특징 |
|---|---|
| List | 순서 O, 수정 O |
| Tuple | 순서 O, 수정 X |
| Set | 순서 X, 중복 X |
| Dictionary | Key-Value 구조 |

# 4. 리스트(List) ⭐⭐⭐ 매우 중요

### 중요도
⭐⭐⭐⭐⭐

### 정의
> 데이터를 순서 있게 저장하는 가변 시퀀스

### 특징
- 순서 O
- 수정 가능
- 추가/삭제 가능
- 여러 타입 저장 가능

### 예시

```python
body = [181, 78, "Dark Brown", "Black"]
```

### 시험 포인트
- 가변(Mutable)
- 다양한 타입 저장 가능
