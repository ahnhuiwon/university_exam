# 30. Pythonic한 카운팅 방식 ⭐⭐⭐ 중요

### 중요도
⭐⭐⭐⭐⭐

기존 방식:

```python
if word in counts:
    counts[word] += 1
else:
    counts[word] = 1
```

Pythonic 방식:

```python
counts[word] = counts.get(word, 0) + 1
```
