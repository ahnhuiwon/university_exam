# 3강 - 파이썬 시작하기

## 1. 데이터 타입(Data Type)

### 중요도
⭐⭐⭐⭐⭐

### 왜 중요한가?
- 프로그래밍의 가장 기초 개념
- 시험에서 무조건 나오는 핵심 개념

### 핵심 정리
데이터 타입은 데이터가 어떤 종류인지 구분하는 역할을 한다.

### 종류

| 타입 | 설명 | 파이썬 타입 |
|---|---|---|
| 정수 | 소수점 없는 숫자 | int |
| 실수 | 소수점 있는 숫자 | float |
| 문자열 | 문자들의 집합 | str |

### 시험 포인트
- 문자열은 반드시 따옴표 사용
- `"123"`은 문자열
- `123`은 숫자

## 3. 연산자(Operator)

### 중요도

⭐⭐⭐⭐⭐

### 핵심 정리

* 피연산자에 대해 특정 연산 수행

### 대표 연산자

| 연산자 | 의미  |
| --- | --- |

| -   | 뺄셈  |
| *   | 곱셈  |
| /   | 나눗셈 |

---

## 4. 특수 산술 연산자 ⭐ 중요

### 중요도

⭐⭐⭐⭐⭐

### 시험 포인트

| 연산자  | 의미   | 예시           |
| ---- | ---- | ------------ |
| `**` | 거듭제곱 | `2 ** 10`    |
| `//` | 몫    | `5 // 2 = 2` |
| `%`  | 나머지  | `25 % 7 = 4` |

### 핵심 정리

* `%`는 짝수/홀수 판별에 매우 많이 사용

```python
num % 2
```

* 결과

  * 0 → 짝수
  * 1 → 홀수

4. 덧셈/뺄셈

### 시험 포인트

## 15. pow() 함수 ⭐ 중요

### 중요도

⭐⭐⭐⭐⭐

### 시험 포인트

```python

```

→ 2³ = 8

```python

```

→ √2

### 핵심 정리

* 0.5 제곱 = 루트
