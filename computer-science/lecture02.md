# 2강 - 파이썬의 이해

## 1. 파이썬의 탄생

### 중요도
⭐⭐⭐⭐

### 왜 중요한가?
- 파이썬의 철학과 방향성을 이해하는 핵심 배경
- 시험에서 창시자/목적/특징 연결 문제 가능

### 핵심 정리
- 개발자: 귀도 반 로썸(Guido van Rossum)
- 1989년 개발 시작
- 크리스마스 휴가 중 취미 프로젝트로 시작
- 이름은 "몬티 파이썬" 코미디 프로그램에서 따옴
- 뱀(Python)과는 직접 관련 없음

### 시험 포인트
- 파이썬은 원래 교육용 언어가 아니라
  분산 운영체제 "아메바(Amoeba)"의
  쉘 스크립트 언어로 개발됨

---

## 2. 파이썬 발전 과정

### 중요도
⭐⭐⭐⭐⭐

### 핵심 정리

| 연도 | 내용 |
|---|---|
| 1991 | Python 0.9 공개 |
| 2000 | Python 2.0 출시 |
| 2008 | Python 3.0 출시 |
| 2020 | Python 2 지원 종료 |

### 시험 포인트
- Python 3.x는 Python 2.x와 비하위 호환
- 인터넷 코드가 안 돌아가면 2.x 문법일 가능성 있음

---

## 3. 파이썬의 철학 (Zen of Python)

### 중요도
⭐⭐⭐⭐⭐

### 왜 중요한가?
- 파이썬 문법과 스타일의 핵심 철학
- 강의에서 매우 길게 설명

### 핵심 철학
- 아름다운 것이 추한 것보다 낫다.
- 명확한 것이 모호한 것보다 낫다.
- 단순한 것이 복잡한 것보다 낫다.
- 복잡한 것이 난해한 것보다 낫다.

### 핵심 정리
- 파이썬은 "가독성"을 매우 중요시함
- 간결하고 읽기 쉬운 코드 지향

---

## 4. 파이소닉(Pythonic)

### 중요도
⭐⭐⭐⭐
