## Q4. 범용 통계패키지

### 문제

다음 중 범용 통계패키지에 대한 설명으로 잘못된 것은?

1. SPSS는 GUI 환경 아래에서 통계분석 및 자료처리가 이루어지므로 쉽게 분석처리를 할 수 있다.
2. SAS는 방대한 양의 자료 처리 기능이 뛰어나며 다양한 통계 분석 절차를 제공하고 있다.
3. 파이썬은 미국에서 개발된 프로그램 언어로 스프레드시트 형태의 데이터 입력을 취하고 있다.
4. R은 객체지향 프로그래밍 언어로서 대화형 통계분석과 그래프 기능이 뛰어나다.

### 정답

**3번**

### 해설

#### SPSS

* GUI 기반
* 메뉴 클릭 방식
* 초보자도 쉽게 사용 가능

#### SAS

* 대용량 데이터 처리
* 기업 및 연구기관 활용

#### Python

* 범용 프로그래밍 언어
* 스프레드시트 형태 입력 방식은 Python의 특징이 아님

#### R

* 통계 분석 특화
* 그래프 기능 우수
* 객체지향 프로그래밍 지원

### 문제

컴퓨터를 통하여 데이터 정보를 효율적으로 처리하기 위해 요구되는 사항이 아닌 것은?

1. 컴퓨터를 쉽게 사용할 수 있어야 한다.
2. 정보를 검색하고 수집·분석하기 위해서는 반드시 컴퓨터 프로그램 언어를 알아야 한다.
3. 데이터 분석에 대한 기본개념과 방법론들을 숙지하여야 한다.
4. 데이터 분석을 위한 다양한 소프트웨어의 활용법을 숙지하여야 한다.

**2번**

### 해설

데이터 분석을 위해 필요한 요소는 다음과 같다.

* 데이터 분석 기본 개념

## Q4. 측정수준

### 문제

다음 측정 수준에 대한 설명 중 옳지 않은 것은?

1. 명목척도에서 각 조사단위에 부여된 숫자는 구분을 목적으로 부여된 기호에 불과하다.
2. 섭씨온도, 습도, 지능지수 등은 구간척도로 측정된 값이다.
3. 변수의 측정수준이 적합한 통계 분석 기법의 선택에 영향을 미치지는 않는다.
4. 계란을 크기에 따라 대·중·소로 구분하면 순서척도로 측정된 것이다.

### 정답

**3번**

### 해설

| 보기 | 판단 |
| -- | -- |
| 1  | O  |
| 2  | O  |
| 3  | X  |
| 4  | O  |

측정수준은 통계 분석 방법 선택에 큰 영향을 미친다.

### 척도 정리

| 척도   | 예시             |
| ---- | -------------- |
| 명목척도 | 성별, 혈액형, 전공    |
| 순서척도 | 만족도, 등수, 대·중·소 |
| 구간척도 | 섭씨온도, IQ       |
| 비율척도 | 나이, 키, 몸무게, 소득 |

## Q5. 데이터 오류 점검

### 문제

데이터를 입력한 후, 제대로 입력되었는지 점검하고자 한다.

데이터의 오류를 점검하기 위한 방법이 아닌 것은?

1. 각 변수의 입력 범위를 벗어난 케이스를 확인한다.
2. 변수 간 논리적 연관성을 고려하여 확인한다.
3. 각 케이스를 조사된 설문지와 비교하면서 점검한다.
4. 설문항목 중 하나라도 응답하지 않은 경우가 있다면 해당 설문지는 분석에서 제외한다.

### 정답

**4번**

### 해설

데이터 오류 점검 방법

* 입력 범위 확인
* 논리적 일관성 확인
* 원본 설문지와 대조

결측치(Missing Value)가 있다고 무조건 설문지를 폐기하지는 않는다.

## Q6. 명목척도 개수

### 문제

통계·데이터과학과 학생을 대상으로 다음 변수를 조사하였다.

| 변수                                    |
| ------------------------------------- |
| 학생의 소속 지역대학                           |
| 학교 생활만족도 (아주 만족, 만족, 보통, 불만족, 아주 불만족) |
| 수강 신청한 과목 중 제일 좋아하는 교과목               |

명목척도로 측정된 변수는 몇 개인가?

### 정답

**3번 (2개)**

### 해설

| 변수       | 척도   |
| -------- | ---- |
| 소속 지역대학  | 명목척도 |
| 학교 생활만족도 | 순서척도 |
| 좋아하는 교과목 | 명목척도 |

따라서 명목척도는 총 2개이다.

### 척도 비교표

| 척도   | 대표 예시                |
| ---- | -------------------- |
| 명목척도 | 성별, 혈액형, 전공, 지역, 교과목 |
| 순서척도 | 만족도, 선호도, 등수         |
| 구간척도 | 온도, IQ               |
| 비율척도 | 나이, 키, 몸무게, 소득       |


SPSS, Excel 등의 도구는 프로그래밍 언어 없이도 사용할 수 있다.
