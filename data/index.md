## Q1. 일반적인 데이터 분석 절차

### 문제

( a ) - 조사, 실험의 계획 - 데이터의 수집 - ( b ) - 분석결과의 평가

( ) 안에 순서대로 가장 적합한 것은?

1. a : 문제의 정의, b : 설문지 평가
2. a : 설문지 작성, b : 데이터의 정리, 분석
3. a : 전문가와의 상담, b : 데이터의 분석
4. a : 문제의 정의, b : 데이터의 정리, 분석

### 정답

**4번**

### 해설

일반적인 데이터 분석 절차는 다음과 같다.

1. 문제의 정의
2. 조사·실험의 계획
3. 데이터의 수집
4. 데이터의 정리 및 분석
5. 분석 결과의 평가

## Q3. 지식정보화사회

### 문제

오늘날과 같이 효율적인 정보의 수집, 수집된 정보의 가치판단, 정보활용 능력이 필수적으로 요구되는 사회를 가장 잘 나타낸 말은?

1. 유비쿼터스사회
2. 전자정보화사회
3. 지식정보화사회
4. 전자통신사회

### 정답

**3번**

### 해설

지식정보화사회는 단순한 정보 보유가 아니라 정보의 수집, 분석, 평가, 활용을 통해 새로운 가치를 창출하는 능력이 중요한 사회를 의미한다.

## Q4. 범용 통계패키지

### 문제

다음 중 범용 통계패키지에 대한 설명으로 잘못된 것은?

1. SPSS는 GUI 환경 아래에서 통계분석 및 자료처리가 이루어지므로 쉽게 분석처리를 할 수 있다.
2. SAS는 방대한 양의 자료 처리 기능이 뛰어나며 다양한 통계 분석 절차를 제공하고 있다.
3. 파이썬은 미국에서 개발된 프로그램 언어로 스프레드시트 형태의 데이터 입력을 취하고 있다.
4. R은 객체지향 프로그래밍 언어로서 대화형 통계분석과 그래프 기능이 뛰어나다.

### 정답

**3번**

### 해설

#### SPSS

* GUI 기반
* 메뉴 클릭 방식
* 초보자도 쉽게 사용 가능

#### SAS

* 대용량 데이터 처리
* 기업 및 연구기관 활용

#### Python

* 범용 프로그래밍 언어
* 스프레드시트 형태 입력 방식은 Python의 특징이 아님

#### R

* 통계 분석 특화
* 그래프 기능 우수
* 객체지향 프로그래밍 지원

## Q5. 데이터 분석을 위한 요구사항

### 문제

컴퓨터를 통하여 데이터 정보를 효율적으로 처리하기 위해 요구되는 사항이 아닌 것은?

1. 컴퓨터를 쉽게 사용할 수 있어야 한다.
2. 정보를 검색하고 수집·분석하기 위해서는 반드시 컴퓨터 프로그램 언어를 알아야 한다.
3. 데이터 분석에 대한 기본개념과 방법론들을 숙지하여야 한다.
4. 데이터 분석을 위한 다양한 소프트웨어의 활용법을 숙지하여야 한다.

### 정답

**2번**

### 해설

프로그래밍 언어를 알면 도움이 되지만 반드시 알아야 하는 것은 아니다.

데이터 분석을 위해 필요한 요소는 다음과 같다.

* 컴퓨터 활용 능력
* 데이터 분석 기본 개념
* 분석 방법론 이해
* 통계 및 분석 소프트웨어 활용 능력

SPSS, Excel 등의 도구는 프로그래밍 언어 없이도 사용할 수 있다.
