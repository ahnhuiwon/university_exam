| 변수                                    |
| ------------------------------------- |
| 학생의 소속 지역대학                           |
| 학교 생활만족도 (아주 만족, 만족, 보통, 불만족, 아주 불만족) |
| 수강 신청한 과목 중 제일 좋아하는 교과목               |

**3번 (2개)**

## Q15. 표본분산

### 문제

다음 중 표본분산을 구하는 공식은?

### 정답

**4번**

### 해설

표본분산 공식

```text
s² = Σ(Xi - X̄)² / (n - 1)
```

계산 과정

1. 평균 계산
2. 데이터 - 평균
3. 편차 제곱
4. 모두 더함
5. n-1로 나눔

### 암기

| 통계량    | 공식                  |
| ------ | ------------------- |
| 평균     | ΣXi / n             |
| 표본분산   | Σ(Xi - X̄)² / (n-1) |
| 표본표준편차 | √s²                 |

## Q16. 변동계수(Coefficient of Variation)

### 문제

자료의 개수나 측정단위가 다른 두 개 이상의 자료에 대한 표본집단 간의 상대적인 산포를 비교할 때 이용되는 측도로, 두 집단의 단위가 다르거나 평균의 차이가 클 때 두 그룹의 산포를 비교하는 데 유용하게 이용되는 측도는?

1. 중앙값
2. 표준편차
3. 표준오차
4. 변동계수

### 정답

**4번**

### 해설

변동계수(CV)는 상대적인 산포 정도를 나타내는 측도이다.

공식

```text
CV = (표준편차 / 평균) × 100
```

### 예시

| 집단 | 평균  | 표준편차 | 변동계수 |
| -- | --- | ---- | ---- |
| A  | 10  | 2    | 20%  |
| B  | 100 | 10   | 10%  |

표준편차는 B가 크지만 상대적인 산포는 A가 더 크다.

### 보기 분석

| 보기   | 의미     |
| ---- | ------ |
| 중앙값  | 중심 위치  |
| 표준편차 | 절대적 산포 |
| 표준오차 | 평균의 오차 |
| 변동계수 | 상대적 산포 |

### 시험 암기

```text
단위가 다르다
평균 차이가 크다
상대적 산포 비교

→ 변동계수(CV)
```
